The Stepping Feet Illusion: Will we notice the same effect with different shapes?

Robert Law and Camila Miranda-Llovera

The illusion we were inspired by is the “stepping feet” illusion. The stepping feet illusion follows the motion of two rectangles that are moving across a grating. While the actual color of the rectangles does not matter, what does matter is that one is lighter in color than the other. The effect is initially observed with a bright yellow and a deep blue, but the same effect is shown when the rectangles are red and green or white and black (the same color as the gratings).

The illusion looks like the rectangles are not moving at the same speed. It appears as though they are moving in alternate times where the yellow rectangle moves, stops, and while it is stopped the blue rectangle will move, thus creating this “stepping feet” motion. However, if one removes the gratings, it can be noted that the rectangles are in fact moving at the same speed. If the contrast of the gratings is lowered, the rectangles move at the same speed and if the rectangles are made to be the same color, the observer can see that they are moving simultaneously. There is also a different effect depending on the number of gratings as well. If there is an odd amount of them, we see a worm-like movement in the two shapes moving across the screen where they appear to contract and expand. If there is an even count of gratings, we see the stepping motion.

The effect that we note is due to the grating in the back. The light color of the rectangle seems to merge with the light grating, thus creating no motion cue and creating the perception that there is no motion. We notice the same effect with the darker rectangle and the dark grating except at alternate times because of the alternating grating.

However, what if we used a different shape? Would we still observe the same effect if we use a shape without the straight edges of a rectangle leading into the straight lines of the gratings? What if we used a rounded edge? We first begin with the same initial design where two rectangles of the same size but different colors (dark blue and yellow) are moving at the same speed. We include different spatial frequencies of 2, 4, 5, 7, 8, 9 so that we can note the stepping and worm-like motions of the shapes. Other variations include using circles, right triangles, and a six-sided L-shape.

The metrics for the gratings are described by how many stripes span the length of the rectangle. Six bars per foot indicate that six gratings of black and white make up the length of the rectangles, eight bars per foot indicate eight bars make up the length of the rectangles, and so on.

Our hypothesis prior to making the shapes was that we would notice the same effect for all the shapes. We thought that since the shapes are all moving into either light or dark gratings, that we would continue to see an effect that is similar to losing the motion cue that we noted for the rectangles.
